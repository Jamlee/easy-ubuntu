# panel

> Ubuntu 主机管理方案

[![Build Status](https://travis-ci.com/Jamlee/panel.svg?branch=master)](https://travis-ci.com/Jamlee/panel)

简洁的 Ubuntu 管理控制面板，简化常见的操作流程

## Usage

```
# 启动前端开发服务
lein fig -- -b dev -r 

# 编译项目为 Jar 包
lein fig -- -bo dev; lein ring uberjar

```
